![demo](https://user-images.githubusercontent.com/19617305/43871780-31103536-9bb9-11e8-9c82-1ec7b2c75c8f.gif)
### 説明
- lumen(PHPフレームワーク)でシンプルなAPIサーバーを構築。
- フロントエンドはReact/Reduxを使用。
- CSSフレームワークはMaterial-UIを使用
- 開発環境にはDockerを使用
### Dockerで使用しているコンテナ
- nginx
- php-fpm
- MySQL
- node
### Docker-compose   
`docker-compose up -d`
