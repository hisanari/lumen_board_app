### 掲示板アプリ
![demo](https://user-images.githubusercontent.com/19617305/43756691-c27f5066-9a50-11e8-9fb1-ab348368472a.gif)
### 説明
- lumen(PHPフレームワーク)でシンプルなAPIサーバーを構築。
- フロントエンドはReact/Reduxを使用。
- CSSフレームワークはMaterial-UIを使用
### Dockerで使用しているコンテナ
- nginx
- php-fpm
- MySQL
- node
### Docker-compose   
`docker-compose up -d`
